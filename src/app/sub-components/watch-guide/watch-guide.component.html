Guide:

Play:

By clicking the play button the tutorial video will begin to play and the code will type out as the tutorial creator typed it.

Jump:

Use the video scrubber to jump to any point of the tutorial and see what the creator did at that time of the tutorial.


Preview:

Click the Preview button or right click any file and preview to see the result of the code at that point in the video.

Download:

Click the Download Project button to download the finished project for modification in your own IDE.

Edit:

Click Edit in Sandbox to change the project directly in the website and preview after
